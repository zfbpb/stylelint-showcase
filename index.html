Code

Inline code should look distinct from prose, while code blocks should handle overflow, padding, and syntax contrast. 

curl https://api.example.com/v1/customers \
  -u sk_test_123: \
  -d email="jenny.rosen@example.com" \
  -d name="Jenny Rosen"
async function createCustomer() {
  const response = await fetch('/api/customers', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      email: 'jenny.rosen@example.com'
    })
  });

  if (!response.ok) {
    throw new Error('Request failed');
  }

  return response.json();
}
